# count_codes
count python lines and show info


usage: 统计代码行数信息

统计行数信息

optional arguments:
  * -h, --help            show this help message and exit
  * -s SORT, --sort SORT  指定key排序
  * -r, --reverse         是否逆序,默认是
  * -f FILE, --file FILE  输入文件名
  * -p PATH, --path PATH  输入文件夹地址


use like:
`
python count_lines.py -p test -r
`
